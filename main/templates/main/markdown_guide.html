Markdown Guide

Mataroa supports Markdown in the blog posts body. Markdown is a very simple formatting language.

Italics

You can use *asterisks* between words to make them italics.

Bold

You can use **double asterisks* to make words bold.

Image

You can write it like this:
![image description - red dot](http://example.com/red-dot.jpg)
to have an image.

Links

Write a link like this: [website link](https://sourcehut.org/)
for it to appear like this website link.

Monospace font

For a monospace font `use backticks` and it will appear like this.


Home | Ethics | Source code
