Comments Guide

Mataroa supports comments on a blog post level. Comments are optional and can be enabled blog-wide from the settings page.

  • Comments are optionally eponymous.
  • There are no configuration options for comments other than enabling them.
  • Blog authors get an email notification when there is a new comment in any of their posts.
  • Commenters do not get any notification, in any case.

  • All comments are held for moderation.
  • The comment review process can take up to 24 hours.
    • {% else %}
  • There is no moderation on comments. Unfortunately, if a blog gets shared somewhere public very capable spammers will not hold back.
