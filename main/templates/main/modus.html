Modus operandi

Details on the what and how of our mode of operation.

Ethics

We are committed to:

  • No tracking of user or visitor behaviour.
  • Never sell any user or visitor data.
  • No ads — ever.

Data privacy

  • Mataroa does not require any personal details in order to be fully functional (email is optional).
  • We support and require encryption in transit via the TLS protocol (versions 1.2 and 1.3).
  • We do not implement data encryption at rest.
  • All rights under GDPR are exercisable
    • via the site
    • by emailing admin@mataroa.blog

Technical

  • Hosted in Helsinki by Hetzner Online GmbH.
  • Daily database backups in AWS Ireland.
  • User passwords are stored SHA256-hashed using PBKDF2.
  • Developed publicly on sr.ht and GitHub.
